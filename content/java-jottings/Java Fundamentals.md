-   Java is an  Objected-Oriented Programming.
-   Java is platform-independent.
-   Java is a strongly-typed language.
-   The main Method is the entry point of a Java program.
-    Java Basic Syntax 
-   Inheritance allows a subclass to inherit the fields and methods of a superclass.
-   Polymorphism allows objects to be treated as instances of their superclass or any of their subclasses.
-   Encapsulation is the process of hiding the implementation details of an object.
-   Abstraction is the process of identifying the essential characteristics of an object and ignoring the irrelevant details.
-    Exception handling is used to handle errors that occur during the execution of a program.
